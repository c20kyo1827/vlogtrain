Vlog Train Code - Introduction

Vlog Train Code provide some collection of simple digital circuit design exercises for practicing. The supported language is Verilog.

Verilog is a Hardware Description Language(HDL). Follow the problem sets or do your simulation by other materials in playground. You can improve your design skills.

Vlog Train Code - How to use
  1. Choose an exercise in the Problem set
  2. Write your design in Verilog
  3. Submit, simluate and debug if necessary

If you want to track you progress, please click the Login/Register to check your status.
